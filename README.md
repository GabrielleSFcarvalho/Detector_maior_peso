# Detector_maior_peso
Exercício prático em Python para praticar lógica de programação, funções, laço for e estruturas condicionais.

# Como funciona
Programa em Python que solicita o nome e o peso de 5 pessoas
e exibe quem possui o maior peso.

## Conceitos utilizados
- Funções
- Laço for
- Estrutura condicional
- Entrada e saída de dados
- Limpeza de tela com módulo os

Projeto desenvolvido para fins de estudo.
